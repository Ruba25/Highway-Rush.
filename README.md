## Game Features

### Scoring System
- **Continuous scoring**: 100 points per second for survival
- **Dodge bonus**: 50 points for each car you successfully dodge
- **Level progression**: Advance every 300 points

### Pause System
- Click the pause button in the top-right corner
- Use keyboard shortcuts (SPACE or P)
- Resume with a 3-second countdown (3... 2... 1... GO!)

### Traffic System
- Cars spawn in 4 lanes with smart collision avoidance
- Minimum spacing maintained between vehicles
- Realistic traffic flow without overlapping
